#####以下是 PHP 二进制文件（即 php.exe 程序）提供的命令行模式的选项参数，您随时可以通过 PHP -h 命令来查询这些参数。

```
php -v     显示当前php版本
php -m     显示当前php加载有效模块
php -i     输出无html格式的phpinfo
php --ini  输出当前php加载ini配置文件的数量、路径信息
php -f     解析和执行 例：php -f  test.php
php -r     运行php代码不加 <?php ?>
```
